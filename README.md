# StorageProduct

## 1. Сборка проекта
Скачать проект из репозитория. Зайти в корень проекта и открыть консоль. В консоле прописать команду "clean install" для сборки проекта.<br>
    `mvn clean install` <br>
Архив war лежит в директории target.
## 2. База данных
